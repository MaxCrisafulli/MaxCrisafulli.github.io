Introduction

HTML documents are made up of text that will appear on screen inbetween tags that give the text certain properties. Tags are often start with an opening tag <exampletag> and end with a closing tag </exampletag>

<!DOCTYPE html>

HTML documents should begin with <!DOCTYPE html> at the top of the page, which signifies that it is a html document

<html>

The next tage should be the <html> tag, it acts as a container for all the other html code (other than the <!DOCTYPE> tag.

     It appears on a html document as this,
     <!DOCTYPE>
     <html>
     (html code)
     </html>

<header>

The header tag is used to denote more important information on a html document that will appear as visible text. Headers typically come at the beginning of a document and will house important links and headings. The header tags look like <header> and </header>, with the information housed between the two tags. Header tags come inbetween the body tags.

<title>

The <title> and </title> tags are used to create the title element which gives the document that name that will appear in the internet browser. In the source code of this page in the meta data there is a title tag that says <title>HTML-Reference</title> The title name should be between the two opening and closing tags.

<body>

The body element (given by the tags <body> and </body>) are used to define where the visible text of the html document will be, they come after the metadata and html tags. The body gives no visible rendering, but what comes between it will appear as visible text on a page

Lists

The list element can be used to create both ordered and unordered lists of items in a html document. Unordered lists are given by the tag <ul> and </ul> with the list items placed inbetween enclosed within in <li> and </li> tags. For example, an unordered list would look like such:

       <ul>
        <li> List Item </li>
        <li> List Item </li>
        <li> List Item </li>
       </ul>
Ordered lists are created in a similiar format, except the tags <ol> and </ol> are used. For example, 
       <ol>
        <li> List Item 1 </li>
        <li> List Item 2 </li>
        <li> List Item 3 </li>
       </ol>
The two lists above will appear as such in a html webpage:
Ordered List:
  1. List Item 1
  2. List Item 2
  3. List Item 3
Unordered List:
  • List Item
  • List Item
  • List Item
