Introduction

HTML documents are made up of text that will appear on screen inbetween tags that give the text certain properties. Tags are often start with an opening tag <exampletag> and end with a closing tag </exampletag>

Lists

The list element can be used to create both ordered and unordered lists of items in a html document. Unordered lists are given by the tag <ul> and </ul> with the list items placed inbetween enclosed within in <li> and </li> tags. For example, an unordered list would look like such:

       <ul>
        <li> List Item </li>
        <li> List Item </li>
        <li> List Item </li>
       </ul>
Ordered lists are created in a similiar format, except the tags <ol> and </ol> are used. For example, 
       <ol>
        <li> List Item 1 </li>
        <li> List Item 2 </li>
        <li> List Item 3 </li>
       </ol>
The two lists above will appear as such in a html webpage:
Ordered List:
  1. List Item 1
  2. List Item 2
  3. List Item 3
Unordered List:
  • List Item
  • List Item
  • List Item

Tables

The table element is used to create tables of data that will appear as visible on the html webpage. The syntax of the table element consists of the following:

  <table> (opening table tag)
   <tr> (top row tag)
    <th>Heading 1</th> (heading item 1 of top row)
    <th>Heading 1</th> (heading item 2 of top row)
    <th>Heading 1</th> (heading item 3 of top row)
   </tr> (top row end tag)
   <tr> (first row start tag)
    <td>Row 1,1</td>
    <td>Row 1,2</td>
    <td>Row 1,3</td>
   </tr> (first row end tag)
   <tr> (second row start tag)
    <td>Row 2,1</td> 
    <td>Row 2,2</td>
    <td>Row 2,3</td>
   </tr> (second row end tag)
</table> (ending table tag)
